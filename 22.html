System warstwowy (Layered system)

W interfejsach API REST wywołania i odpowiedzi przechodzą przez różne warstwy. Pętla komunikacyjna może obejmować pewną liczbę elementów pośredniczących. Interfejsy muszą być zaprojektowane w taki sposób, aby ani klient, ani serwer nie miał informacji o tym, czy komunikuje się z aplikacją końcową, czy elementem pośredniczącym. (mozliwosc dodania load-balancery / proxy / bezpieczenstwo / wydajnosc)
