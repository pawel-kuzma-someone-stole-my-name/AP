Status Codes
200 OK wszystko ok :)
201 CREATED Nowy zasób został stworzony
204 NO CONTENT Serwer pomyślnie przetworzył żądanie i nie zwraca żadnej zawartości
304 NOT MODIFIED Wskazuje, że zasób nie został zmodyfikowany od wersji określonej przez nagłówki żądania If-Modified-Since lub If-None-Match. W takim przypadku nie ma potrzeby retransmitowania zasobu, ponieważ klient nadal posiada wcześniej pobraną kopię
400 BAD REQUEST Serwer nie może lub nie chce przetworzyć żądania z powodu czegoś, co jest postrzegane jako błąd klienta (np. nieprawidłowa składnia żądania, nieprawidłowe obramowanie wiadomości żądania lub zwodnicze kierowanie żądania).
401 UNAUTHORIZED Podobny do 403 Zabronione, ale przeznaczony do stosowania, gdy wymagane jest uwierzytelnienie, które nie powiodło się lub nie zostało jeszcze dostarczone.
403 FORBIDDEN Serwer zrozumiał żądanie, ale odmawia jego autoryzacji. Serwer, który chce upublicznić, dlaczego żądanie zostało odrzucone, może opisać ten powód w treści odpowiedzi (jeśli istnieje)
404 NOT FOUND Serwer źródłowy nie znalazł aktualnej reprezentacji dla zasobu docelowego lub nie chce ujawnić, że taka istnieje
422 UNPROCESSABLE ENTITY Serwer rozumie typ zawartości encji żądania (stąd kod statusu 415 Unsupported Media Type jest niewłaściwy), a składnia encji żądania jest poprawna (stąd kod statusu 400 Bad Request jest niewłaściwy), ale nie był w stanie przetworzyć zawartych w niej instrukcji
